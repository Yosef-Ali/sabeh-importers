# Sabeh Importers — Marketplace Platform

E-commerce and import management platform built for Ethiopian businesses. Product catalog, inventory display, and customer inquiry system.

**Tech Stack:** Next.js · TypeScript · Tailwind CSS · PostgreSQL · Supabase

## Features

- Product catalog with categories and search
- Import shipment tracking and management
- Customer inquiry and order system
- Responsive design with Ethiopian market focus
- Admin dashboard for inventory management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
